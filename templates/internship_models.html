Machine Learning Models

Machine Learning models can be understood as a program that has been trained to find patterns within new data and make predictions.These models are represented as a mathematical function that takes requests in the form of input data, makes predictions on input data, and then provides an output in response.

PREDICTION

DECISION TREE

A decision tree is a non-parametric supervised learning algorithm, which is utilized for both classification and regression tasks. It has a hierarchical, tree structure, which consists of a root node, branches, internal nodes and leaf nodes.

PREDICTION

SUPPORT VECTOR MACHINE

Support Vector Machine(SVM) is a supervised machine learning algorithm used for both classification and regression. Though we say regression problems as well its best suited for classification.

PREDICTION

RANDOM FOREST

Random forest is a Supervised Machine Learning Algorithm that is used widely in Classification and Regression problems. It builds decision trees on different samples and takes their majority vote for classification and average in case of regression

PREDICTION

K-NEAREST NEIGHBOUR

The k-nearest neighbors algorithm, also known as KNN or k-NN, is a non-parametric, supervised learning classifier, which uses proximity to make classifications or predictions about the grouping of an individual data point.

PREDICTION

NAIVE BAYES

The Naive Bayes classification algorithm is a probabilistic classifier. It is based on probability models that incorporate strong independence assumptions. The independence assumptions often do not have an impact on reality.
